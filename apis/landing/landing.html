// the pattern

spot a specific pain. build the smallest tool that fixes it. ship.

no marketplaces. no platforms. no signup walls. each wedge gets its own subdomain, its own demo, its own honest scope. more wedges land when the pain is real.

// the apimesh way

built for the way real devs hit real walls.

standalone

each wedge has its own URL and demo. no umbrella signup, no platform tax.

honest scope

one pain, one tool. if it grows past that, it becomes a new wedge.

cli + mcp

works in your terminal and in your agent. no bespoke clients.

open source

conway is public on github. read the code, fork the wedge, ship your own.
